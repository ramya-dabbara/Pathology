# Pathology

Starter scaffold for a Pathology module UI with reusable layout blocks.

## What is ready

- Vite + React + TypeScript app structure
- Sidebar and Header placeholders
- Router path for Orders page
- Folder layout similar to your previous project
- Devops-related folder intentionally not created

## Copy-paste targets from old project

- Sidebar code: `src/components/layout/Sidebar/Sidebar.tsx`
- Header code: `src/components/layout/Header/Header.tsx`
- Orders page code: `src/pages/orders/OrdersPage.tsx`
- Common layout wrapper: `src/components/layout/DashboardLayout.tsx`
- Global styles: `src/styles/global.css`

## Run locally

```bash
npm install
npm run dev
```
